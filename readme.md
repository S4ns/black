# Installation
```powershell
# clone the repo
$ git clone https://github.com/S4ns/black.git

# change the working directory to sherlock
$ cd black

# install the requirements
$ python3 -m pip install -r requirements.txt
```

# Usage
```
$ python3 black.py --help
usage: black.py [-h] -u USERNAME [-d DATETIME] [-l]

optional arguments:
  -h, --help   show this help message and exit
  -u USERNAME  username of target
  -d DATETIME  select snapshot by datetime
  -l           list snapshots of target
```
